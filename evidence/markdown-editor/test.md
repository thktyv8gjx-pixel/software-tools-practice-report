# 我的第一个Markdown文档

## 什么是Markdown

Markdown是一种**轻量级标记语言**，可以通过简单符号编写结构清晰的文档。

### Markdown的核心特点

1. **简洁性**：使用少量标记完成排版；
2. **可读性**：未渲染的纯文本仍然容易阅读；
3. **兼容性**：可用于GitHub、Hexo和技术文档。

### 常用工具

| 工具 | 主要作用 |
| --- | --- |
| VS Code | 编辑Markdown文档 |
| Hexo | 将Markdown构建为静态网页 |
| Git | 记录文档版本 |
| Nginx | 发布生成的静态网站 |

### JavaScript代码示例

```javascript
function hello() {
    console.log("Hello, Markdown!");
}

hello();
```

> Markdown源文件既便于人工阅读，也便于Git进行版本差异比较。
